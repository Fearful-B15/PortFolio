Garden - E-commerce webpage

Garden is an e-commerce website for a garden store. It features a responsive design, product listings, and a shopping cart functionality.

Overview

The main goal of Garden is to create a seamless user experience where gardeners can interact with the website in a natural way. Whether they need to find the best gardening tools, understand the basics of gardening, or get personalized suggestions, Garden serves as an interactive tool to assist them.

Garden Webpage Features

  • Different pages for each part of the webpage (Gardening Plants and tools,Pet friendly plants)
  • Contact form for easy contacting between customer and business to ensure there is communication between user and person
  • Filter products feature to showcase products with specific tags

Technology Stack

Technologies Used: HTML, CSS, JavaScript, PHP.
