Georage Bakery Website

Georage Bakery is a fully functional bakery website that includes a product catalog, a shopping cart, and an intuitive checkout system. I created this website as part of an assignment, with a focus on user experience and responsive design. The website allows customers to easily browse products, add them to their cart, adjust quantities, and proceed to checkout. The cart functionality is powered by JavaScript, which dynamically updates prices based on user selections.

Features

  • Product catalog with a range of bakery items
  • Custom shopping cart to manage orders
  • Dynamic pricing and order updates using JavaScript
  • Mobile-friendly and responsive design
  • Secure checkout process (mock implementation)

Technology Stack

Technologies Used: HTML, CSS, JavaScript, Responsive Design
